⬇ 下载 Windows 版

一分钟内完成安装。

1. 下载 SmoothScroll_0.1.15_x64-setup.exe

2. 双击安装。无需管理员密码。

3. SmoothScroll 自动启动。
   在系统托盘附近找到图标。

右键点击托盘图标打开设置。
按 Ctrl+Alt+S 切换平滑滚动。

安装到 %LOCALAPPDATA%。无需管理员权限。

1. 下载 SmoothScroll_0.1.15_aarch64.dmg (Apple Silicon)

2. 打开 .dmg，将 SmoothScroll 拖到应用程序。

3. 从应用程序打开 SmoothScroll。
   在提示时授予辅助功能权限。

macOS 需要辅助功能权限才能拦截滚动事件。

有问题？有答案。

如何在 Windows 11 上启用平滑滚动？

下载 SmoothScroll 并安装，平滑滚动将自动在系统范围内启用。无需配置。

如何在 Windows 上获得 Mac 风格的惯性滚动？

SmoothScroll 拦截鼠标滚轮输入并应用平滑缓动曲线，给您与 macOS 相同的动量滚动体验。

SmoothScroll 免费吗？

是的。完全免费，采用 MIT 许可证开源。无付费版本，无广告。

支持游戏鼠标吗？

支持。适用于任何发送标准滚轮信号的鼠标，包括罗技 G 系列、雷蛇、MX Master 等。

与 WizMouse 有何不同？

在输入驱动层工作，效果最顺滑，支持按应用排除、120 Hz，二进制文件仅 4 MB。

安全吗？反作弊系统呢？

安全。不注入游戏也不修改游戏内存，与 EAC 和 BattlEye 等反作弊系统兼容。

设置存储在哪里？

存储在您的用户配置文件夹中。不发送任何数据 — 无遥测。

支持 Linux 吗？

目前仅支持 Windows 和 macOS。Linux 桌面环境通常已内置平滑滚动。
