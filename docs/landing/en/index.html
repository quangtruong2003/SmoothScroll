Questions? Answers.

How do I enable smooth scrolling on Windows 11?

Download SmoothScroll, install it, and smooth scrolling is automatically enabled system-wide. No configuration required.

How do I get Mac-style inertia scrolling on Windows?

SmoothScroll intercepts your mouse wheel input and applies a smooth easing curve, giving you the same momentum-based scrolling feel as macOS.

Is SmoothScroll free?

Yes. SmoothScroll is completely free and open source under the MIT license. No tiers, no ads, no nagging.

Does SmoothScroll work with gaming mice?

Yes. SmoothScroll works with any mouse that sends standard scroll wheel input, including Logitech G-series, Razer DeathAdder, MX Master, and most other mice.

How is SmoothScroll different from WizMouse?

SmoothScroll works at the input driver level for the smoothest possible result, with per-app exclusion, 120 Hz support, and a lightweight 4 MB binary.

Is it safe? What about anti-cheat?

SmoothScroll is safe. It does not inject into games or modify game memory, so it is compatible with anti-cheat systems like EAC and BattlEye.

Where are settings stored?

Settings are stored in your user profile folder. No data is sent anywhere — there is no telemetry.

Does SmoothScroll work on Linux?

SmoothScroll is currently Windows and macOS only. For Linux, native desktop environments often have built-in smooth scrolling.
