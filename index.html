SmoothScroll

Smooth mouse-wheel scrolling for Windows 10/11 and macOS.

Native low-level input interception · frame-perfect easing · per-app exclusion · free & open source.

Download latest release View on GitHub

What it does

Most mice emit discrete wheel ticks that feel jagged in apps without native inertia. SmoothScroll sits between the OS and your applications, swallows raw wheel events, and re-emits them as fluid, eased pulses at 120 Hz — giving you Mac-style smooth scrolling on Windows and consistent inertia across every app on macOS.

Features

  • Native input interception — Windows low-level mouse hook (WH_MOUSE_LL); macOS CGEventTap.
  • Frame-perfect easing — Linear, Cubic, Quintic, Exponential curves tuned for 120 Hz output.
  • Per-app exclusion — opt apps out by process name; everything else stays smoothed.
  • Global hotkey — Ctrl+Alt+S default, rebindable.
  • System tray UI — left-click for settings, right-click for quick controls.
  • Auto-update — signed releases via Tauri updater (opt-in).
  • Tiny footprint — single Rust binary, no Electron, no background services.

Install

Windows 10 / 11

Download SmoothScroll_<version>_x64-setup.exe (NSIS) or .msi from the Releases page. Per-user install, no admin rights.

macOS 12+ (Apple Silicon)

Download the .dmg, drag SmoothScroll.app to /Applications, right-click → Open on first launch (Gatekeeper), then grant Accessibility access.

FAQ

How do I enable smooth scrolling on Windows 11?

Install SmoothScroll, launch it, and scrolling becomes smooth system-wide. No drivers, no admin rights, no reboot. Toggle on/off with Ctrl+Alt+S.

How do I get Mac-style inertia scrolling on Windows?

SmoothScroll re-emits raw mouse-wheel ticks as eased pulses at 120 Hz, producing the same gliding inertia feel as macOS trackpad scrolling — for any wheel mouse, in any Windows app.

Does it work with Logitech MX Master, Razer, and other gaming mice?

Yes. SmoothScroll intercepts at the OS level, so any mouse the OS recognizes works.

Is it safe with anti-cheat (EAC, BattlEye)?

SmoothScroll uses the same standard input APIs as accessibility tools and screen readers. Per-app exclusion lets you disable it for games that prefer raw input.

Is it free?

Yes — MIT licensed, no telemetry, no ads.
